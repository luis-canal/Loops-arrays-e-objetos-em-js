# 🔄 Loops, Arrays e Objetos

Repositório com exercícios e exemplos práticos de JavaScript focados em estruturas de repetição, manipulação de arrays e uso de objetos.

## 📚 Conteúdos abordados

- Loops (`for`, `while`, `for...of`)
- Arrays
- Métodos de array
- Objetos
- Manipulação de dados
- Lógica de programação

## 🛠️ Tecnologias utilizadas

- JavaScript

## 🎯 Objetivo

Praticar conceitos fundamentais de JavaScript para fortalecer a base em programação e resolução de problemas.
